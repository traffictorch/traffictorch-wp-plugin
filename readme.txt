Instant access to Traffic Torch AI, GEO & SEO tools directly from the Gutenberg editor sidebar.

== Description ==

Traffic Torch WP adds a clean panel in the Gutenberg editor sidebar for every post and page.

Select any AI, GEO or SEO tool and launch it with one click — the current preview or live URL is automatically passed to the tool.

All analysis is performed on **traffictorch.net**, which is hosted on Cloudflare's secure global edge network. 
The plugin itself is lightweight and does not send any data from your WordPress site — it only opens the chosen tool in a new browser tab with the URL pre-filled.

Features:
- Native Gutenberg sidebar widget with one-click tool launcher.
- Educational admin dashboard with tool overviews and help guides.
- Recommended lightweight plugins to complement Traffic Torch.
- Fully respects WordPress dark mode and is mobile-friendly.

No server load on your site. No tracking. No data collection by the plugin.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. The "Traffic Torch" menu will appear in the admin sidebar.
4. While editing any post or page, look for the Traffic Torch panel in the Document sidebar.

== Frequently Asked Questions ==

= Does this plugin send any data to Traffic Torch? =
No. The plugin only opens a new browser tab to traffictorch.net and passes the URL of the post/page you are editing. 
All analysis happens in your browser communicating directly with traffictorch.net (hosted on Cloudflare's secure edge network). 
No data is sent or stored by the plugin itself.

= Where does the actual analysis happen? =
The heavy lifting (audits, AI analysis, etc.) is done on traffictorch.net, which runs on Cloudflare Pages, a fast, secure, and globally distributed platform.

= Is the plugin completely free? =
Yes, the core plugin is 100% free and open source. Traffic Torch Pro (optional paid plan on traffictorch.net) offers higher limits and advanced features.

= Does it work with the Classic Editor? =
No. The editor sidebar panel requires the Block Editor (Gutenberg).

= Can I use it on client sites? =
Yes. The plugin is licensed under GPLv2 and can be used on unlimited sites.
